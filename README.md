# LitCode

Free LeetCode-to-GitHub sync with clean README notes.

LitCode saves accepted LeetCode submissions to GitHub and adds the part LeetCode usually keeps behind Premium: time complexity, space complexity, and a compact growth graph.

## Highlights

- Accepted solutions sync automatically to your chosen GitHub repo.
- Each problem gets a clean `README.md` with runtime, memory, time complexity, and space complexity.
- Complexity is shown with Big-O notes and a small x/y growth chart.
- Local code analysis powers the README notes, so no external AI API key is required.

## Setup

```bash
npm install
npm start
```

Copy `src/config.example.js`, then set your GitHub OAuth values.

## Build

```bash
npm run build
npm test
```
